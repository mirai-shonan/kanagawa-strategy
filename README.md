# チームみらい 神奈川活動戦略

チームみらいの神奈川県内での活動戦略をデータに基づいてまとめたページです。

## 公開ページ

https://mirai-shonan.github.io/kanagawa-strategy/

## 概要

令和8年2月衆院選・令和7年7月参院選の神奈川県開票データをもとに、
市区町村別の得票率・票数を分析し、街頭活動・ポスティングの戦略をまとめています。

## 内容

- **データ分析**：概況・全自治体一覧・衆参比較・当選議員
- **戦略**：コスパ優先エリア・伸びしろエリア・イベントカレンダー
- **ターゲット別**：学術・若者 / 子育て / 祖父母世代

## ファイル構成

```
index.html         # メインHTML（コンテンツ・タブ構造）
style.css          # CSS（デザイン・レイアウト）
app.js             # JSロジック（テーブル描画・グラフ・タブ切替等）
data.js            # データ定義（CONFIG, DATA, POSTING, KAWAI）
data-template.js   # 他県用データテンプレート（Fork時に使用）
README.md          # このファイル
```

## 他の都道府県への横展開（Fork方式）

このリポジトリをForkして、以下の手順で各都道府県版を作成できます。

### 手順

1. **このリポジトリをFork**する
2. **`data.js` を差し替える**
   - `data-template.js` をコピーして `data.js` にリネーム
   - CONFIG（都道府県名・選挙データ・政党別得票率など）を入力
   - DATA（市区町村別の得票データ）を入力
   - POSTING（ポスティング実績データ）を入力
   - KAWAI（候補者別データ、不要なら空配列）を入力
3. **`index.html` のコンテンツを編集**
   - タイトル（`<title>`タグ）を変更
   - 各戦略セクション（コスパ・イベント・学術・子育て・シニア等）の内容を自県に合わせて書き換え
   - ヘッダー・概況・テーブル・グラフ等のデータ表示部分は `app.js` が `data.js` から自動生成するため編集不要
4. **GitHub Pages を有効化**して公開

### 注意点

- `app.js` と `style.css` は共通ロジック・デザインなので基本的に編集不要
- データ表示（概況の数字、テーブル、グラフ）は `data.js` の内容から自動生成される
- 戦略コンテンツ（イベント情報、重点エリア解説等）は各県固有のため `index.html` 内で直接編集
- `data-template.js` の各フィールドにコメントで説明あり
